# Save it🎮

A Scratch game built for CS50x where you protect a character from flying objects.

![Game Preview](preview.png)

## Play it live
👉 [Click here to play on Scratch](https://scratch.mit.edu/projects/1329083121)

## How to play
- Click **Bombs** before they hit your character
- Let **Hearts** reach your character for +1 Life
- Let **Stars** reach your character for bonus points
- Clicking a good object = −5 points penalty
- Game speeds up over time — survive as long as you can!

## Built with
- Scratch 3.0
- Course: CS50x — Introduction to Computer Science (Harvard / edX)

## Key features
- Custom block with input parameter: `setup clone (type)`
- Cloning system for all flying objects
- Broadcast messages, variables, and difficulty scaling
